# Erra Space Studio Website

Static website for **Erra Space Studio** (Вишгород, Україна), hosted on GitHub Pages.

## Live site
[https://erraspace.com.ua](https://erraspace.com.ua)

## Structure
- `index.html` — головна сторінка
- `style.css` — стилі
- `script.js` — логіка перемикання мов та модальні вікна
- `images/` — зображення для напрямків

##  Features
- Двомовність (UA/EN)
- Модальні вікна для опису напрямків
- Адаптивна сітка для карток
- Google Analytics (GA4) інтеграція

##  Технології
- **HTML5 / CSS3 / JavaScript**
- **GitHub Pages** для хостингу
- **Lazy loading** для оптимізації зображень
- **Sitemap.xml + robots.txt** для SEO


## SEO та індексація
- Додано `sitemap.xml` для швидкої індексації
- Додано `robots.txt` для відкритої індексації
- Підтвердження домену у Google Search Console (через TXT‑запис або мета‑тег)
